# Code Comment

## 说明

```
TODO: 标识处有功能代码待编写

FIXME: 标识处代码需要修正，甚至代码是错误的，不能工作，需要修复

XXX: 标识处代码虽然实现了功能，但是实现的方法有待商榷，希望将来能改进

OPTIMIZE: 标示处代码可能是低效的，待优化

HACK: 标识处代码需要根据自己的需求去调整程序代码
```
